form

무언가를 입력, 선택, 자료를 전송하게만드는 기능을가진 요소

양방향 전송방식 - 입력 - 보내기 - 받아서 처리 - 결과를 피드백

검색, 로그인, 회원가입, 구매, 게시판

  1. 검색하기 위해 필요한 단어/문장을 작성(글을 작성할 수 있는 영역)
  2. 전송버튼을 클릭(a는 data를 보내는 능력x, data보내는 능력을 가진 버튼)
  3. 데이타를 전송받을곳으로 전송(어디로 보낼지 체크, 공개(get)/비공개(post).....)
  4. 전송받은 곳에서 무언가 수행(backend)
  5. 수행에 대한 결과를 피드백
  6. 피드백 받은 결과를 도출
각 영역별 기능 구분
